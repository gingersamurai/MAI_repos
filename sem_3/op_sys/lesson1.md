## системный вызов

- `int id = fork()`
- `int x = fact(5)`

> ***fork()*** — системный вызов в Unix-подобных 
операционных системах, создающий новый процесс (потомок), 
который является практически полной копией 
процесса-родителя, выполняющего этот вызов.
